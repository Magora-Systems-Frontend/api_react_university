# React Sample App

## Run the application locally

### Dependencies
- Node v10+
- cloned repository

### Running dev

```bash
after cloning the repo:

# install packages
$ npm i

# to run server
$ npm run json-server

# to run dev 
# head over to http://0.0.0.0:8020
$ npm run dev


```

### Running prod

```bash
after cloning the repo:

install dependencies
npm i --production

to run prod client and server
head over to  http://0.0.0.0:8022
$ npm run build


```
